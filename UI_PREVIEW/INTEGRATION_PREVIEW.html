▲ Vercel Webhook 配置
将此地址填写到 Vercel 项目设置 → Webhooks → Add Webhook
📖 Vercel Webhook 配置步骤：
1. 前往 Vercel Dashboard → 项目 → Settings → Webhooks
2. 点击 "Add Webhook"，Payload URL 填入上方地址
3. Secret 字段填入创建集成后生成的 Webhook Secret（用于 HMAC-SHA1 签名验证）
4. 勾选 Deployment 相关事件，保存
🚂 Railway Webhook 配置
将此地址填写到 Railway 项目设置 → Webhooks
📖 Railway Webhook 配置步骤：
1. 前往 Railway Dashboard → 项目 → Settings → Webhooks
2. 点击 "Add Webhook"，填写上方地址
3. 选择要监听的事件类型（DEPLOY_SUCCESS / DEPLOY_FAILED / SERVICE_CRASH）
4. 保存后，Railway 将向此地址推送部署事件
